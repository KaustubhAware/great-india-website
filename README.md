# 🇮🇳 The Great India Website

A creative website built using **HTML and CSS** to showcase the greatness of India, including famous personalities, scientists, and freedom fighters who contributed to the nation.

---

## 🌐 Live Website

Visit the live project:

https://kaustubhaware.github.io/great-india-website

---

## 📌 Features

* Beautiful **Hero Section with Indian theme**
* **Gallery of Great Personalities**
* Sections for **Freedom Fighters and Scientists**
* **Responsive card layout**
* **Tables for important personalities**
* **Lists and formatted text**
* **External links for more information**
* Embedded **iframe section**

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
